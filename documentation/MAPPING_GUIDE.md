Excel Column	PPT Shape Name	Description
A (Col 1)	EntityName	Main Title of the slide
B (Col 2)	Status	Subtitle or Status Indicator

D-AR (44 Cols)	Custom_01...	Various entity metadata
Setup Instructions

Open Excel and press Alt + F11.

Go to Tools > References and check Microsoft PowerPoint 16.0 Object Library.

Import the .bas files.

Ensure Template.potx is in the same folder as the Excel file.
